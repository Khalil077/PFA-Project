# PFA Project

**Tech Stack:** Angular (frontend) + NestJS (backend)

## Project Structure

- `angular-frontend/` → The Angular frontend
- `nest-backend/` → The NestJS backend

## Description

This is a full-stack project:
- Frontend built with Angular
- Backend built with NestJS
